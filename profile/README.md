# 🌲 Paper Tree Studios

**Crafting immersive, real-time JRPG experiences and the technical infrastructure that supports them.**

Welcome to the official GitHub home of **Paper Tree Studios**. We are dedicated to the intersection of stylized game design and high-performance multiplayer systems.

---

## 🏗️ Active Projects

### 🐍 [pypapertree](https://github.com/Paper-Tree-Studios/pypapertree)
**The Core Engine.**
A real-time MMOJRPG engine built in Python. Featuring a server-authoritative combat system, spatial audio, and a persistent biome-based overworld.

*   **Technologies:** Python, Pygame-CE, WebSockets, NumPy, SQLite.
*   **Focus:** Authoritative networking, spatial acoustics, and geometric skill scaling.

### 🌐 [papertree-web](https://github.com/Paper-Tree-Studios/papertree-web)
**The Studio Showcase.**
Our primary web presence and landing page. Designed to provide a deep-transparency, parallax-driven environment that mirrors the aesthetic of our games.

*   **Technologies:** Next.js, React, Framer Motion, Tailwind CSS.
*   **Focus:** Visual storytelling, interactive media showcase, and deployment orchestration.
