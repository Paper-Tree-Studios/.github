# 🌲 Paper Tree Studios

**Making fun games together to share with other people.**

Welcome to **Paper Tree Studios**. We are a development organization dedicated to the craft of immersive, real-time multiplayer experiences and the technical infrastructure that supports them.

---

## 🏗️ Active Projects

### [pypapertree](https://www.papertreestudios.org/games)
The core MMOJRPG engine. Built with Python, WebSockets, and Pygame-CE. Featuring authoritative combat, biome-based persistence, and a living overworld.

### [papertree-web](https://www.papertreestudios.org)
Our proprietary Studio Showcase and digital home. Enjoy our real-time parallax environment and dynamic time-of-day system.
